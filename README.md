# pulsate-dev/short-url

pulsate.dev short url.

## Routes

| Endpoint     | Redirects to                              |
| ------------ | ----------------------------------------- |
| `/discord`   | Pulsate Discord Server                    |
| `/github`    | Pulsate GitHub                            |
| `/x`         | Pulsate X                                 |
| `/youtube`   | Pulsate YouTube                           |
| `/community` | Pulsate Community (GitHub Org Discussion) |
